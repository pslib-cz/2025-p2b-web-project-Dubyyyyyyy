1. ROOT

box-sizing: border-box: Nastavíš pro všechny prvky (*), aby padding nezvětšoval celkovou šířku elementu.

2. BODY    

margin: 0: Odstraníš výchozí bílý okraj prohlížeče.

background-color: Nastavíš barvu pozadí (např. tmavou pro moderní vzhled).

font-family: Zvolíš hlavní písmo.

font-size: Nastavíš základní velikost textu (obvykle 16px).

color: Barva písma (musí být v kontrastu k pozadí).

3. .CONTAINER

max-width: Omezíš šířku obsahu (např. 1200px).

margin: 0 auto: Vycentruješ celý blok na střed obrazovky.

padding: Nastavíš vnitřní okraj, aby se text na mobilu nelepil na hrany displeje.

4. DISPLAY

display: grid nebo flex: Hlavní nástroje pro skládání prvků.

grid-template-columns: Rozdělíš prostor na sloupce (např. repeat(3, 1fr)).

gap: Nastavíš přesné mezery mezi kartami nebo sekcemi.

5. H1, P, A
font-size: Nadpisy (h1) dostanou větší váhu, texty menší.

line-height: Zvýšíš řádkování (např. 1.6), aby se text lépe četl.

text-decoration: none: U odkazů odstraníš podtržení.

margin-bottom: Uděláš prostor pod odstavci, aby nebyly nahuštěné.

6. .BUTTON, .CARD

background-color: Barva tlačítek.

padding: Vytvoříš vnitřní prostor tlačítka (např. 10px 20px).

border-radius: Zakulatíš rohy (např. 8px).

transition: Nastavíš plynulost animace (např. 0.3s ease).

7. :HOVER

transform: scale(): Lehké zvětšení prvku.

filter: brightness(): Tlačítko se při najetí může mírně rozsvítit.

cursor: pointer: Aby uživatel věděl, že na prvek lze kliknout.

8. @MEDIA

flex-direction: column: Změníš řazení prvků z řady do sloupce.

font-size: Na mobilu můžeš zmenšit hlavní nadpisy, aby nezabíraly celou obrazovku.
